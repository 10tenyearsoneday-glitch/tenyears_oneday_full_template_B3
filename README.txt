TEN YEARS ONE DAY（十年一日）— 靜態網站模板（Canva 風格）

✅ 功能
- 分頁式導覽（不會把內容堆在同一頁下面），使用 #hash 路由
- 全系列 / 純銀飾品：商品卡、款式、數量、加入購物車、圖片放大
- 購物車：查看選購內容、刪除、改數量、前往結帳
- 結帳：需登入會員，配送方式：7-11 / 全家 / 宅配；自動計算運費與折扣
- 會員：手機 + 密碼登入；註冊後自動進入資料頁；可看訂單編號與明細
- 折扣：首購 9 折、當月壽星 85 折、滿額免運（門檻/運費/折扣可後台調整）
- 後台：完全內建（localStorage）可新增/修改/刪除商品、刪除訂單、調整公告與免運門檻
- 隱藏管理員入口：左下角完全隱形（可 tab 到）

⚠️ 重要提醒（安全）
此模板是「純前端」示範：帳密與資料在瀏覽器 localStorage，無法防止惡意使用。
若要正式收款、真正會員資料、7-11/全家串接、Stripe/綠界等，必須接後端。

📌 使用方式（GitHub Pages）
1) 把整個資料夾內容上傳到 repo 根目錄（不要再包一層資料夾）
2) Settings → Pages → Source：Deploy from a branch → Branch: main / root
3) 等待 Pages build 完成後即可開站

📌 使用方式（Netlify）
- Build command 留空
- Publish directory 填「.」（根目錄）
- 直接拖放整個資料夾即可
